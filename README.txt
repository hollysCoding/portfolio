This is a trading application that buys and sells stocks as well as keeping track of the user's portfolio up to five individual stocks. The application provides current prices of stocks and keeps track of the users balance to keep the user informed.

File needed to compilation: 
1. project3main.c
2. trading.h
3. tradingApp.h
4. common.h
5. simulator.h
6. project3main.lib


Compilation Instructions:

gcc -o project4withlib_004.exe tradingApp.c common.c simulator.c trading.c project3main.c project3main.lib

Execution Instructions: 
The file "mystocks.txt" as well as "myportfolio.txt" must be in the folder of the executable. 

Execute by   ./trading

Enjoy the application! =)
